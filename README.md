# Python Calculator

A simple Python calculator that performs basic arithmetic operations.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⚡ Power

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/1krish8008/Python--calculator-
   ```
2. Run the calculator:
   ```bash
   python calculator.py
   ```

## Usage
- Enter the first number
- Enter the second number
- Choose an operation (+, -, *, /, **)
- View the result
- Choose to calculate again or exit
